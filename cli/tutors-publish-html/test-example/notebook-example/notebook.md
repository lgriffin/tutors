# Python Data Analysis Notebook

This is an interactive Python notebook that demonstrates basic data analysis concepts using pandas and matplotlib.

## What you'll learn

- Basic Python data structures
- Working with pandas DataFrames
- Creating visualizations with matplotlib
- Data cleaning and manipulation

## Getting Started

Click the "Run All" button to execute all cells in the notebook. You can also run individual cells by clicking on them and pressing Shift+Enter.

The notebook environment includes:

- **Python 3.9** - Full Python standard library
- **NumPy** - For numerical computing
- **Pandas** - For data manipulation
- **Matplotlib** - For creating visualizations
- **Seaborn** - For statistical data visualization

## Interactive Features

- **Persistent Storage**: Your work is saved in the browser
- **Full Jupyter Interface**: All the features you know from Jupyter Lab
- **No Server Required**: Everything runs in your browser using PyOdide
